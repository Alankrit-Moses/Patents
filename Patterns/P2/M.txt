Let E = {e1, e2, ..., en} be a set of comparable entities and let m be an attribute measured over time. Let v(e, m, t) denote the value of entity e for attribute m at time t.

For a shared interval [t_start, t_end], a growth divergence pattern holds when there exist two entities ei and ej such that both entities are measured using the same attribute, aggregation method, start time, and end time, and their absolute gap increases meaningfully over the interval:

|v(ei, m, t_end) - v(ej, m, t_end)| > |v(ei, m, t_start) - v(ej, m, t_start)| + epsilon

where epsilon >= 0 is a threshold for a meaningful increase in separation.

Equivalently, define the pairwise gap between ei and ej at time t as:

gap(ei, ej, m, t) = |v(ei, m, t) - v(ej, m, t)|

Then growth divergence holds when:

gap(ei, ej, m, t_end) - gap(ei, ej, m, t_start) > epsilon

All compared entities must use the same attribute, aggregation method, start time, and end time. Entities with missing start or end values cannot be used to establish the pattern.
